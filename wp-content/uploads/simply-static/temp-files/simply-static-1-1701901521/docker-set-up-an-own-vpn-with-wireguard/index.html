My Raspberry PI Projects

  • Docker: Set up an own vpn with Wireguard

    In this Project I want to configure a vpn service form my raspberry pi, I will use wireguard running in Docker, I can use wireguard application installed in any device, the device will be linked to my raspberry network reading the QR published inside the code once the container is running.

    Docker hub:

    https://hub.docker.com/r/linuxserver/wireguard

    Wireguard google play:

    https://play.google.com/store/apps/details?id=com.wireguard.android&hl=es&gl=US

    Updating and installing dependences in the raspberry:

    1. sudo apt-get update
    2. sudo apt-get upgrade
    3. sudo apt-get install docker
    4. sudo apt-get install docker-compose
    5. sudo nano docker-compose.yml

    The docker-compose file can be found in the link below:

    https://github.com/usefitsoftware/docker_wireguard

    sudo docker-compose up -d

    with sudo docker ps we can check the containers we have running:

    The QR code can be used throw the wireguard app installed.

    sudo docker-compose logs

    Docker commands:

    https://docs.docker.com/engine/reference/commandline/docker/
